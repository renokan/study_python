## Flask OVDP - Сайт по ОВГЗ (фреймворк Flask)

**Особенности реализации:**
- Вся логика в одном файле *app.py** (без мейнов, вьювов и блюпринтов).
- Без базы данных, данные по аукционам хранятся в файле **db_auctions.json**.
