Отчет по аукционам ОВГЗ

Что анализируем: Результати розміщення облігацій внутрішніх державних позик.

Источник данных: API (в формате json) Национального банка Украины (НБУ).

Что хотим узнать: Сколько денег получено (за год) в рамках аукционов и сколько нужно вернуть (за год).

Чего нет в цифрах: Не анализируются выплаченные проценты, т.е. суммируем за год все значения "Залучено коштів від розміщення облігацій" для даты "розміщення облігацій" (год) и эту же сумму для даты "термін погашення облігацій" (год).

Для аукционов в национальной валюте (грн) период начинается с 2009 года, в оригинале с 2001 года.

Описание терминов: "In" - получено денег; "Out" - нужно вернуть; billion - млрд. грн; million - млн. USD/EUR.
